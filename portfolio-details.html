Farmer Portal

Project information

  • Client: Internal Project
  • Project date: 01 June, 2021
  • Project URL: www.farmerportal.com

NGO Donor

Project information

  • Category: Charity Management System
  • Client: Internal Project
  • Project date: 01 jan , 2021
  • Project URL: www.ngodonorsystem.com

"NGO Donor System" is a comprehensive platform designed to streamline the process of connecting donors with non-governmental organizations (NGOs). This system simplifies the donation process, enhances transparency, and helps NGOs manage their resources more efficiently. By facilitating direct communication between NGOs and potential donors, the system aims to improve the effectiveness of charitable contributions and ensure that help reaches those in need promptly. The system includes features such as document verification, event management, donation tracking, and detailed reporting to enhance accountability and donor engagement.

Dgital Notice Board

Project information

  • Category: Digital Notice Board System
  • Client: Internal Project
  • Project date: 01 June, 2018
  • Project URL: www.digitalnoticeboard.com
  • Languages Used: Python, JavaScript, HTML, CSS

"Digital Notice Board" is a software application designed to provide a centralized control for displaying notices across multiple screens at various locations. The system allows for the uploading of event photos, event details, and event dates. Notices can be displayed according to specific courses and will have both start and end dates, after which they will be automatically removed. This digital solution stores notices, photos, and news clippings, making it easy to retrieve last year's event photos, news clippings, and notices. The centralized control ensures efficient management of notices and enhances communication within an institution.
